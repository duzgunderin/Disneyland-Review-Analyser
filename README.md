#Disneyland Review Analyzer

## Project Overview

This project is a Python application that analyses Disneyland review data from a CSV dataset. The system allows users to view review information, generate charts, and export summary data.

The project was developed using Python and demonstrates key programming concepts including functions, loops, conditionals, file handling, visualizations, and Object-Oriented Programming.

---

## Features

- Load review data from a CSV file
- View reviews for a selected park
- Count reviews by park and reviewer location
- Calculate average scores
- Export summary data to a CSV file
- Display visual charts using Matplotlib
- Demonstrate Object-Oriented Programming with a Review class


---

## Files Included

- 'main.py'  >  Main program fow
- 'process.py' > Data processing functions
- 'tui.py'  >    Text-based user interface
- 'visual.py' >  Visualisation charts
- 'review.py' >  Review class
- 'disneyland_reviews.csv' > Dataset file
- 'exported_summary.csv' > Exported summary output
